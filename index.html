My Projects

Institutional Website for a Digital Company

In this group project, we created a complete institutional website for a major digital company, aimed at Generation Alpha—specifically 3rd-year middle school students exploring career paths and internships. Our work included researching the company’s structure, services, and digital/ecological transition, then translating that into accessible, engaging content. We designed mockups, gathered user needs, and developed a functional, eco-conscious website with clear navigation and simplified language tailored to a non-specialist audience.

Debian Server Setup & Web Stack Deployment

As part of a systems administration module, I set up a minimal Debian 12 server using QEMU/KVM virtualization, configured without a graphical interface for performance and security. I installed and configured key web technologies, including Apache, PostgreSQL, PHP, and PhpPgAdmin. The project included verifying network connectivity, managing services via the command line, setting up SSH access with port forwarding, and securing PostgreSQL with hashed credentials. This hands-on experience strengthened my skills in Linux system administration, virtualization, and deploying web application infrastructure.

Road Accident Data Analysis

In this data-driven project, we analyzed over 15 years of French road accident records from the national open data repository. Working in a team of two, we cleaned, transformed, and explored the dataset using PostgreSQL and statistical tools like R. The goal was to answer a specific public-interest question through visual data storytelling and insightful charts. We addressed data quality issues (missing values, inconsistencies), handled multi-year schema variations, and applied basic project management methods (risk assessment, timeline planning). This project offered a first real-world experience in data science workflows, from raw data to meaningful public communication.

Content Classification - Machine Learning

This project focuses on developing an automatic classification system for news articles. The objective was to assign each news snippet to a category based on its content. In the first phase, classification was done using manually constructed lexicons with weighted keywords. In the second phase, I implemented a machine learning-inspired method to generate these lexicons automatically by analyzing word frequency and specificity across categories. The project highlights my skills in natural language processing, algorithm design, and object-oriented programming in Java.
